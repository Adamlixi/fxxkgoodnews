# 🤪 好事网

一个抽象气息的新闻网站, 弗洛伊德·峰点评新闻。

## 🎭 特色功能

- **新闻锐评**：争取每天更新
- **Meme风格**：彩虹渐变背景 + Comic Sans字体 + 沙雕表情包
- **响应式设计**：手机、平板、电脑都能完美显示

## 🚀 快速开始

```bash
npm run dev
# 打开 http://localhost:3000
```

## 🎨 技术栈

- **Next.js 14** - 最新的React框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 原子化CSS
- **Comic Sans MS** - 最沙雕的字体

**警告**：本网站内容极度沙雕，请谨慎浏览，后果自负！
