# 🤪 好事网

一个充满沙雕气息的程序员新闻网站，专门收集各种离谱的编程故事和bug制造现场！

## 🎭 特色功能

- **沙雕新闻**：每天更新最离谱的程序员日常
- **Meme风格**：彩虹渐变背景 + Comic Sans字体 + 沙雕表情包
- **互动体验**：点赞、分享、收藏，让你的快乐加倍
- **分类浏览**：震惊、笑死、离谱、沙雕、无语，总有一款适合你
- **响应式设计**：手机、平板、电脑都能完美显示

## 🚀 快速开始

```bash
npm run dev
# 打开 http://localhost:3000
```

## 🎨 技术栈

- **Next.js 14** - 最新的React框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 原子化CSS
- **Comic Sans MS** - 最沙雕的字体

**警告**：本网站内容极度沙雕，请谨慎浏览，后果自负！
